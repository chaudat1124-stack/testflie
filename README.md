# nhom4-kanbanflow
Ứng dụng quản lý công việc theo mô hình Kanban.

## Chức năng chính:
- Tạo/Xóa Board (Bảng dự án)
- Quản lý Cột (To-do, Doing, Done, v.v.)
- Kéo & Thả Task (Drag & Drop)
- Tìm kiếm & Lọc công việc
- Lưu trữ cục bộ bằng Hive (Hỗ trợ Web/Mobile/Desktop)

## Kỹ thuật sử dụng:
- Flutter & Bloc
- Clean Architecture (Domain, Data, Presentation)
- Repository Pattern
- Unit test Repository
- Hive Storage
- Flutter Animate (Hiệu ứng mượt mà)

## Hướng dẫn cài đặt:
1. Chạy `flutter pub get`
2. Chạy `flutter run`
